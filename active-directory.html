Environnement
Windows Server, Active Directory Domain Services (AD DS), GPO

Description

Déploiement et configuration d'un domaine Active Directory sous Windows Server. Le TP couvre la promotion d'un serveur en contrôleur de domaine, la création du domaine, la jonction de postes clients, la gestion des utilisateurs, groupes et unités d'organisation (OU).

Des stratégies de groupe (GPO) ont également été configurées pour appliquer des paramètres de sécurité et de configuration uniformes à l'ensemble des postes du domaine.

Compétences mobilisées

  • Active Directory (AD DS)
  • Windows Server
  • Gestion des GPO
  • Création d'OUs et groupes
  • Jonction au domaine
  • Gestion des comptes utilisateurs
  • Administration Windows
  • Contrôleur de domaine
