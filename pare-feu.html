Description

Configuration du pare-feu Windows Defender avec fonctions avancées de sécurité (Windows Defender Firewall with Advanced Security). Le TP couvre la création de règles de filtrage pour le trafic entrant et sortant, la gestion des profils (domaine, privé, public) et l'application de politiques de sécurité réseau via GPO.

Compétences mobilisées

  • Windows Defender Firewall
  • Règles entrantes / sortantes
  • Profils de pare-feu
  • Filtrage par port et protocole
  • Administration Windows
  • Sécurité réseau
  • GPO et déploiement de règles
